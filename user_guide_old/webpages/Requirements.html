Requirements Overview

You will need the following to generate and upload a wrapper for your dataset
    Java - Java Runtime Environment will allow you to run Spoon
    Spoon - A full-featured open-source data integration tool

       Just download and extract spoon.zip
       On Windows, just double-click spoon.bat
       On Mac and other Unixes, open a terminal and execute './spoon.sh' or 'sh spoon.sh'
       This will launch our Help Center and the Spoon User Interface.
