Welcome to Help Center!

The Col*Fusion name stands for Collaborative [data] Fusion.
The system is aimed to play a role of interactive data repository which is open for everyone. Everyone is welcome to provide some data in form of datasets (we also call them stories) and utilze the data already available in the system. In addition, the system leverages the crowd to build relationships between the stories.
The help center will help you to get familiar with the system and/or find answers to some questions you might have.

Please choose appropriate category on the left to start learning about the system.
